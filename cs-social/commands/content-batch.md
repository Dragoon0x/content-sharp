# Content Batch

Generate a week of social content from one source piece

## Usage

Run this command when you need to generate a week of social content from one source piece. The command applies the frameworks and best practices from the cs-social skill domain to produce actionable output.

## Process

1. Gather context — understand the specific requirements, audience, and objectives
2. Apply relevant frameworks from the cs-social domain
3. Generate output with practitioner-level quality
4. Review against the quality checklist before delivering

## Output Format

Structured, actionable output ready for immediate use or further refinement.
